Count the number of elements strictly greater than an element X in a non-zero array

Input Format

all the elements of array A
Element X

Output Format

output one line with one integer: the total number of array elements that are greater than X

Example 1:
Input: Array = [1,2,3,1,1,3]
           X = 2

Output: 2

Example 2:
Input: nums = [4,5,3,1,2]
           X = 3

Output: 2
